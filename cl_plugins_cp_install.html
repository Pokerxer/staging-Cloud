    Install and Configure the cPanel Plugin

    task
    The Weebly cPanel plugin allows you to offer Weebly to all your customers seamlessly using cPanel.

    cPanel Requirements

    This cPanel plugin requires:

    • cPanel 11.46 or higher
    • CentOS 6+

    Install the cPanel Plug-in

    You download the plug-in from the Cloud Admin. The cPanel plugin is an rpm file.

    1. From the Plugins tab of the Cloud Admin, click the download icon for cPanel.

    2. Follow the standard rpm installation and update processes, for example:

      sudo rpm -ivvh weeblycpanel-live-1.x.x.noarch.rpm.

    3. Once installation is complete, copy the provided config.ini file to ​/usr/local/cpanel/3rdparty/etc/weebly/config.ini.

    These instructions are also available in the install.txt file found in the zip file.

    Configure the cPanel Plug-in

    Configuration takes place in config.ini. This file lives in /usr/local/cpanel/3rdparty/etc/weebly/. If you don’t see one there, then copy the sample config.ini from the zip file. The configuration file contains your Weebly Cloud API key and secret. If the plugin was downloaded through Cloud Admin, then it should already contain your API key and secret. If you’re using the sample, then you can find your key and secret on the Settings page of the Cloud Admin.
