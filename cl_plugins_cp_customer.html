    cPanel Customer Guide

    concepts
    Learn how your customers will work with Weebly and cPanel

    Now that you’ve installed and configured the cPanel plugin here’s how your customers will interface with it. ​ Just like most cPanel plugins the user can access the Weebly plugin by finding the plugin logo in their cPanel home page.

    After the user clicks the Weebly icon they are sent to a page to start creating their first Weebly site. They will see two things. The first is a brief introduction video that goes into detail about the benefits of a Weebly website. The second is a dropdown of all domain names that they currently have in cPanel and can be used with Weebly. If a domain does not appear, it either contains the word “Weebly” or is already in use on the Weebly platform.

    Once the user clicks the Add button below the dropdown they will be sent to a url (we call this url the billing url) for payment information.

    If you use WHMCS in conjunction with cPanel you should use WHMCS as your billing url. Learn more here.

    Once the user completes the purchase through the billing url they will now see an Edit Site button upon their next visit to cPanel. The Edit Site button gives them direct access to their Weebly website.

    The user can have as many Weebly sites as they’d like. The Add Site button will appear if they have at least one domain that isn’t a Weebly site.
