    AppDirect Customer Guide

    concepts
    Learn how customers work with the AppDirect plug-in.

    Weebly AppDirect Profile

    Customers view AppDirect profiles when browsing the marketplace. A profile provides detailed information on features, pricing, tutorial videos, screenshots and questions for a product, and is what your customer will first see when the click on Weebly from within your marketplace. Here’s the Weebly profile (the full Weebly profile can be viewed appdirect.com).

    ​Customers purchase a Weebly site in your marketplace by clicking the Start a Free Trial button at the top (if you choose to not offer free trials, then the button displays Try Now).

    Creating a New Site

    Once they decide to start a trial or purchase, customers choose the Weebly plan best suited for their site.

    Next, they enter a domain name along with payment information. After these steps are complete they are sent directly into the Weebly editor through a single sign-on link.

    Managing Sites

    Once a site is created, users access the Site Editor from the Weebly tile on the MyApps tab. They can also optionally upgrade the plan offered on a site (if one is available) by selecting Manage App.

    Upgrades

    One of the big advantages to using Weebly is the tight integration in the event a user upgrades while using the Weebly editor. The editor determines if a user has access to a given feature by the Weebly plan assigned to the site. If they try to use a feature not offered by their current plan, they are prompted to upgrade. This screenshot shows a user trying to access the HD Video element when they don’t have access to it.

    Because the site’s current plan doesn’t support HD video, a dialog displays stating that an upgraded plan is required for that element. The user can upgrade immediately, or can opt to “Try it out” without upgrading; however, it will not display on the published site.

    When they click Upgrade Now an upgrade window opens. The user is sent directly to your AppDirect marketplace where the upgrade takes place. Once the upgrade is complete the user automatically has access to the premium feature and can continue building their site.

    Cancellations

    Customers can delete their site and cancel their plan from the Manage App selection.

    Deleting a site takes down the published site and removes future access to the site.
