    Create Bundles in WHMCS

    task
    Along with creating single products from Weebly plans to offer in WHMCS, you can also bundle a Weebly plan with cPanel Shared Hosting in WHMCS.

    You can create bundles with Weebly plans and Shared Hosting managed by cPanel.

    Note: Plans included in bundles cannot be sold separately. You must create plans specifically for the bundle.

    Purchasing a bundled plan that includes Weebly and Shared Hosting automatically provisions a Weebly site for the domain chosen during the ordering process, as well as sets publish credentials for that site. The customer will simply log into their cPanel,navigate to the Weebly area, and quickly access their site editor by clicking the Edit Site button.​

    Create a Plan for a WHMCS Bundle in Weebly

    The first step in creating a bundle is to create a Weebly plan specifically for the bundle, following the process here. When creating the bundle, configure it as you would any other plan created in Weebly.

    This screenshot shows a version of the Weebly Pro plan created specifically for a bundle.

    Import the Bundle Plan into WHMCS

    Once you create the plan in Weebly, you need to import it into WHMCS, as you would anytime you create new plans that need to be synched with WHMCS. Remember to set the ​price to the true value of the plan within its respective bundle. This price will be added to the price of all other products in the bundle to create the final bundle price.

    Hide the Plan for the Bundle

    You need to configure the plan to include in your bundle so that it’s hidden in your list of plans. Open Plan Preferences and select Hide from Order Form.

    Create the Bundle

    Now that the Weebly plan is configured, you can create your bundle.

    1. Hover over the Setup ​tab, scroll down to Products/Services,​ and click on Product Bundles.​
    2. Click Create New Bundle.
    3. Set the Name​ of your bundled offering. This is the name that will display to your customers. Next, designate the offerings date range. It is highly recommended that you select No Expiry so that the bundle does not auto-expire. Select Allow Promotions if you wish to offer this bundle as a promotional offering (discounts, specials, etc.), and select Show in Product Group​ if you want to offer this bundle within a specific category/area of your WHMCS store.
    4. Click Save Changes.

    Your bundle is created and you now need to configure it for sale.

    Configure the Bundle

    You now need to configure the settings for your bundle. Each step is annotated in this image.

    ​

    1. For Bundle Items, select the the products to include in your bundle. Make sure you select the Weebly plan created specifically for this bundle. If you don’t, your customers will not be able to find it, as it is hidden from the store as a standalone product. If you need to, you can click on the Configure link for a bundle item to set the billing cycle or override the price. You can also elect to include the bundle in a product group.

    2. For Product Description, describe all of the products/features a customer would enable by purchasing the bundle. This is the only product description available for the bundled package, so it is important that you are clear and concise wth your description.

    3. Set Display Price to the sum of the individual prices in the bundle. This is not the actual price of the product, but does need to reflect that price. By default, the actual price is determined by adding up the prices of each individual product in the bundle.

    4. Set Sort Order to where in your lineup of offerings you want this bundle to appear.

    5. Click Save Changes.
