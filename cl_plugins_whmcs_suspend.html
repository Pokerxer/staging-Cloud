    Suspend and Reactivate Accounts in WHMCS

    You may need to suspend a userâ€™s access to a site for non-payment or some other reason. When you suspend access to a site, the user can no longer access the editor for that site.

    To suspend a user:

    1. In WHMCS, navigate to the Clients Profile page.

    2. Click the Suspend button.

      This button calls the Weebly APIs to disable editor access for that site. Users with suspended access will not be able to access the editor for that site, but the published site remains fully functional.

    To reactivate a user:

    Use the Unsuspend button to reactivate the user.
