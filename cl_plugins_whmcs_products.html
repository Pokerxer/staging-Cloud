    Set Up Products in WHMCS

    task
    Weebly plans are offered as products in WHMCS

    Import and Sync Weebly Plans

    When you install the WHMCS plug-in, any plans that you have configured in the Weebly Cloud Admin are automatically created as products for you in WHMCS. Going forward if you add new plans or update existing ones, you need to import them into WHMCS.

    To import and sync plans:

    1. In WHMCS, click Plans in the left navigation.
    2. Click Import Plans. A new sync of Weebly plans and WHMCS products occurs.

    Edit Plans in WHMCS

    Plan terms and pricing that are set in Weebly are automatically imported into WHMCS. However, you can override these configurations.

    To override Weebly plans, click the Edit icon on the far right of the plan listing.
