    Install and Configure APS2

    task
    Learn how to install and configure the APS2 plug-in, for use with Odin Service Automation (OSA).

    The Weebly Cloud platform provides a plug-in for APS, for use with Odin Service Automation (OSA). Once you download and install an instance of the Weebly Website Builder APS (WWBA), and configure service templates and plans, your customers will be able to purchase Weebly plans and create and publish websites, as part of a bundle (or on their own). Customers will be able to purchase additional plans or upgrade using the Customer Control Panel.

    This section provides high-level information on the install and setup process. For full details, refer to Service Provider Deployment Guide For Odin Service Automation found in the plug-in zip file.

    Prerequisites

    You’ll need to set up the following before starting your Weebly APS instance install and configuration:

    • Create an admin account in the Weebly Cloud Admin portal.
    • Create an API key and key secret that will be used to communicate between OSA and Weebly.
    • Create Weebly plans in Weebly Cloud Admin.
    • Install OSA version 6 or greater.
    • Download the Weebly Website Builder APS zip file to a local directory.
    • Determine if you will host the published sites or if they will be hosted at Weebly. If you plan on hosting the published sites, then you can take advantage of automating the publication process. Details can be found in the “Prerequisites for Weebly Website Builder Application” section of Odin’s Deployment Guide.
    • Set the Application endpoint (requires php and php-runtime library, 5.3 or later). Follow the “Prepare Provisioning Host” instructions in the Deployment Guide.
    • Meet requirements for Weebly Cloud.

    Import Weebly Website Builder APS and Create an Instance

    You use the Odin Provider Control Panel to import the Weebly Website Builder APS zip file and create an instance. You will need your API key and secret to complete this step. These can be found on the Settings tab of the Weebly Cloud Admin portal.

    Now that you’ve installed and created an instance, you need to configure the instance to work with existing Weebly plans. By creating service templates made up of Plan and other types of resources, you can provide Weebly plans alone or as part of a bundle to your customers.
