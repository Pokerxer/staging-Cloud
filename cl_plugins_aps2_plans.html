    Create Plans in APS2

    concepts task
    You create plans in APS that reflect the different Weebly plans available to sell to your customers.

    After you import Weebly plans, you can then use them to create the service plans that your customer will purchase.

    Import Weebly Plans

    Weebly plans need to be created using the Weebly Cloud Admin portal and then imported into the Weebly Website Builder APS (WWBA). Anytime you make changes to your plans or add new plans, you’ll need to synchronize the WWBA.

    Once plans are imported into WWBA, you add them as resources to be included in service templates.

    Create the Resources for Your Service Templates

    Service templates represent the bundle of services that a customer can buy, and can include any number of Weebly plans and websites, along with any other services you’d like to include. Any service to be included must be first created as a resource. You create Weebly as a resource by creating a “Plan” resource.

    Tip: Remember to sync your Weebly plans in the WWBA if you’ve made any changes in the Weebly Cloud Admin portal.

    Create Service Templates and Service Plans for Your Resources

    A service template creates a package that a customer can purchase. You need to create at least one service template that includes the Weebly resource(s) created in the previous step. You then need to create the service plans and rates for the template to provide the billing info needed to support the Weebly plans. When creating the rates, you can set resource limits and fees for a given service plan. For example, if you want to allow customers to purchase a set number of Weebly plans, you can set that resource limit in the rate.

    Additionally, if you want to allow customers to purchase additional resources and add them to the plan, you can set that up in the resource rates and service plans. Building this into the plan allows customers to purchase additional plans without needing to upgrade.
